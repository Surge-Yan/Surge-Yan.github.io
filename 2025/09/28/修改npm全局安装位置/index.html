修改 npm 的默认全局安装位置为 D 盘而不是 C 盘

一篇面向 Windows 用户的超细避坑指南：路径迁移、权限修复与验证全流程

背景：为什么要把 npm 全局目录搬离 C 盘？

  • C 盘空间日渐告急，而 npm 全局包 + 缓存动辄上 GB。
  • 重装系统时想保留已装工具，独立磁盘更方便。
  • 团队/公司电脑权限受限，D 盘通常更容易写入。
